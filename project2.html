VR Solution for Tele-Collaborative e-Health Applications

The EU-funded TrialsNet project aims to carry out large-scale trials to implement a series of innovative 6G applications based on various technologies.

In this context, Use Case 7 "Remote Proctoring" aims to introduce innovative solutions using intelligent peripherals, such as a telepresence system and VR headset, to support remote proctoring in interventional cardiology.

The aim of this work was to compare, in an experimental context, the standard telepresence system with an innovative system that connects the remote operator to the surgeon via a VR headset, allowing immersive interaction; moreover, adding artificial latencies to verify the effect.



The first phase of the application involves training the surgeon to become familiar with the VR headset. The task requires collaboration with the remote proctor to build figures using Lego bricks. This preliminary exercise helps develop coordination and communication, better preparing the surgeon for the actual task.













The core task involves conducting remote proctoring activities in interventional cardiology, making use of an ultrasound device to support the procedure.
